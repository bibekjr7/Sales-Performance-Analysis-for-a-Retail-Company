# Retail Sales Performance Analysis — Python & Pandas

## Project Overview
An end-to-end exploratory data analysis of a retail sales dataset using Python.
This project covers data cleaning, aggregation, trend analysis, and visualization
to answer key business questions about sales performance.

## Business Questions Answered
1. What is the total revenue across all transactions?
2. Which gender segment drives the most sales?
3. Which product category generates the highest revenue?
4. What does the sales trend look like over time — are there seasonal patterns?

## Key Findings
- Identified the highest revenue-generating product category and customer segment
- Revealed monthly sales trends and patterns over the full dataset period
- Cleaned and validated raw data including duplicate removal and negative value filtering

## Python Concepts Demonstrated
- Data loading and inspection with pandas
- Data cleaning — duplicate removal, negative value filtering, null checking
- Aggregation and grouping with groupby
- Time series analysis using datetime parsing and period grouping
- Data visualization with matplotlib and seaborn
- Automated insight generation (top category, top segment)

## Tools Used
- Python (pandas, matplotlib, seaborn)
- VS Code

## How to Run
1. Clone the repository
2. Install dependencies: `pip install pandas matplotlib seaborn`
3. Place `retail_sales_dataset.csv` in the project folder
4. Run `analysis.py`
